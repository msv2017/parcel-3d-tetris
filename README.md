# parcel-3d-tetris

(WIP) 3D Tetris using only css, html5 canvas, javascript and parcel.

## Build & Run
In command line run
``parcel index.html``

Then open in your browser
``http://localhost:1234``

To move the figure use ``A`` and ``D`` keys.
